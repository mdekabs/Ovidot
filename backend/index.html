Mdcator: Authentication Microservice

Welcome to Mdcator! This microservice provides robust authentication and user management functionalities for your application, enabling seamless integration of secure user authentication into any project. Whether you're developing a web application, mobile app, API, or any other service requiring user authentication, Mdcator simplifies the process and ensures your users' data is protected.

Features

  • User Authentication: Securely authenticate users with username and password.
  • Token-Based Authentication: Generate authentication tokens for secure API access.
  • User Management: Create, update, and delete user accounts with ease.
  • Password Reset: Allow users to securely reset their passwords using one-time passwords (OTP).
  • Swagger Documentation: Explore and test API endpoints effortlessly with Swagger documentation.

Example Integration

Here's a simple example of how to integrate Mdcator into a Node.js application using the axios library:

const axios = require('axios');

// Example sign-in request
axios.post('http://16.171.242.67:5000/sign_in', {
    username: 'example_user',
    password: 'example_password'
})
.then((response) => {
    console.log(response.data);
})
.catch((error) => {
    console.error(error);
});
